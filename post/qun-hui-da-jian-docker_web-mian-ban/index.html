群晖搭建docker_web面板

2023-11-29, 253 words, 1 min read

由于群晖最近docke总是提示注册表查询失败，每次还要使用本地的代理加速才能拉取。搜索教程后特地搭建一个docker_web面板。
1、创建root用户
首先参考博客内此文章WinScp访问群晖根目录创建root用户，一是如果不使用root用户搭建会提示没有权限。二是直接在群晖中搭建好后会提示“监控事件已退出”。

2、拉取容器
使用刚刚创建的root用户账号进入SSH，执行以下命令
docker run --name docker_web
-p 8999:8999
--restart=always
-v /var/run/docker.sock:/var/run/docker.sock
-d joinsunsoft/docker.ui

参数说明：
docker run：启动container
–name：容器命名
–restart=always：自启动
–privileged=true：权限
-p：映射容器的端口到主机上的端口
-v：将容器的目录映射到本地计算机上目录中
-e：参数
-d：在后台运行容器
