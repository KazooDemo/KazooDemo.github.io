群晖阿里云DDNS

2021-12-11, 197 words, 1 min read

  1. 登录阿里云控制台，鼠标右上角头像，点击 “accesskey管理”。

  2. 创建好账号，授权。

  3. 权限管理-授权
    选择就是 管理云解析（DNS）的权限。

  4. 将账号的 AccessKeyID、AccessKeySecret 保存下来。

  5. 域名进去域名解析

    1. 对自己的域名，进行解析的设置；
    2. 选择添加记录；
    3. 记录类型选择类型A；
    4. 主机记录，补充完整自己的域名；
    5. 记录值，添加自己的公网ip。

  6. SSH客户端连接NAS，sudo -i切换到root用户，执行下面代码。

    1. 项目地址：https://registry.hub.docker.com/r/sanjusss/aliyun-ddns/
    2. docker run -d --restart=always --net=host
      -e "AKID=[AccessKeyID]"
      -e "AKSCT=[AccessKeySecret]"
      -e "DOMAIN=ds.97io.cc"
      -e "REDO=30"
      -e "TTL=60"
      -e "TIMEZONE=8.0"
      -e "TYPE=A,AAAA"
      sanjusss/aliyun-ddns
